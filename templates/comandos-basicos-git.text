Principais comandos  básicos do Git

1) Configuração nome de usuário 
git config --global user.name Wfzinn5

2)Configurando o e-mail do usuário
git config --global user.email welrosevania@gmail.com

3) Verificar o usuário configurado
git config user.name

4) Verificar o e-mail configurado
git config user.email

5) Iniciar um repositório 
git init

6)Adicionar os arquivos para o envio do GitHub

git add .
